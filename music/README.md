# Music Files for Wedding Website

## Instructions
Place your wedding music files in this folder.

### Required File
- `wedding-song.mp3` - The background music for your wedding website

### Supported Formats
- MP3 (recommended)
- WAV
- OGG

### Tips
1. Keep file sizes reasonable (under 10MB) for faster loading
2. Choose romantic, soft instrumental music for background ambiance
3. Consider the volume level - the music is set to 30% volume by default
4. Popular wedding song suggestions:
   - "Canon in D" by Pachelbel
   - "A Thousand Years" by Christina Perri (instrumental)
   - "Perfect" by Ed Sheeran (instrumental)
   - "All of Me" by John Legend (instrumental)
   - Custom romantic instrumentals

### How to Add Music
1. Save your chosen wedding song as `wedding-song.mp3` in this folder
2. The music player will automatically detect and play the file
3. Guests can toggle the music on/off using the elegant player button

### Music Player Features
- Floating music button with royal styling
- Play/pause toggle functionality
- Automatic volume control (30% for background ambiance)
- Elegant icons that match the wedding theme
- Responsive design for all devices
